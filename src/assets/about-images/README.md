# About Page Images

This folder contains images for the "A Picture Is Worth a Thousand Words" section on the About page.

## How to Add Images

**Simply add your image files to this folder!** The images will be automatically discovered and displayed in the grid - no code changes needed.

### Supported Formats

- JPG/JPEG
- PNG
- WebP
- GIF

### Naming

You can name your images however you like. They will be automatically sorted alphabetically by filename.

**Examples:**
- `workspace.jpg`
- `design-process.png`
- `image-1.jpg`, `image-2.jpg`, etc.
- Any naming convention you prefer!

### Notes

- Images will be displayed in a 4-column grid on large screens
- Images maintain a 9:16 aspect ratio (portrait orientation)
- All images are automatically optimized by Vite during build
- Add or remove images anytime - they'll automatically appear/disappear from the page
